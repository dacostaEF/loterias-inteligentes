# Scripts de Diagnóstico

Esta pasta concentra exemplos e execuções manuais que antes ficavam em blocos
`if __name__ == "__main__"` dentro de módulos de produção. Use estes scripts
para rodar análises de forma isolada, sem impactar o app.

Exemplos a adicionar gradualmente:
- Análise de distribuição Lotofácil
- Análise estatística avançada Lotofácil

Observação: mantenha dependências de dados (Excel) via paths relativos ao raiz
do projeto ou variáveis de ambiente.
